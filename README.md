## 使い方
------
**懸賞広場のプログラムについて**  
#### 手順
1. seleniumとwebbrowserをpipなどでインストール
2. chromedriverをダウンロード
3. chromedriverのpathを通す
4. アカウントを設定する。
5. プログラムを実行。この時、１つアカウントを設定したなら[0]を入力する
